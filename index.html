Settings

GitHub Integration

Required for adding/removing flights from this interface. Your token is stored in browser localStorage only.

Must be a Classic PAT (not fine-grained) with repo scope.
GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)

Push Notifications (ntfy.sh)

Install the ntfy app on your phone and subscribe to your topic below.
Set the same topic as NTFY_TOPIC in your GitHub repository Secrets.

Subscribe URL: https://ntfy.sh/your-topic

AviationStack API (Optional)

Enables auto-lookup of flight info (times, aircraft, airline) when adding flights.
Free tier: 1,000 calls/month. Get free key →

Also set this as AVIATIONSTACK_KEY in GitHub Secrets for automatic updates
